# HERMS4

**Открытая модульная система автоматизации для пивоварения, дистилляции и ректификации**

HERMS4 — это проект полностью открытой системы управления технологическими процессами малого производства.  
Он объединяет программно-аппаратные решения для:

- варки пива (HERMS-метод);
- дистилляции и ректификации;
- перспективного расширения на сыроварение, теплицы, пасеку и другие цеха.

Архитектура построена по принципу разделения уровней:
- **CBPi4 (Raspberry Pi)** — стратегическое управление, рецепты, интерфейс;
- **Arduino Mega 2560** — тактическое управление в реальном времени (PID, стабилизация мощности, аварийная логика);
- **Промышленные регуляторы (ТРМ151, РЭХ‑100)** — ручной резерв и SIL4‑безопасность.

Проект создаётся как открытая платформа для энтузиастов и малых производств.

## Ключевые особенности

- **HERMS‑пивоварение** с противоточным теплообменником;
- **Ректификация и дистилляция** со стабилизацией мощности;
- **Гибридное управление SSR** (PID / стабилизация);
- **Ручной резерв** на независимых промышленных регуляторах;
- **Многоуровневая ПАЗ** (независимая цепь, термостаты, защита от осушения);
- **Модульная интеграция** с Node‑RED, MQTT, CAN, RS485, LoRa;
- **Открытая документация** и полная воспроизводимость.

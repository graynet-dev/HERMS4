# Техническое задание (ТЗ) на АСУ ТП Цеха 1 (Пивоварня / Винокурня)

## 1. Функциональные возможности (Что система делает?)

### 1.1. Общие цели
Система предназначена для автоматизации процессов варки пива, сбраживания, а также дистилляции и ректификации крепких напитков в условиях цеха малого производства (до 100 литров за варку).

**Основные принципы работы:**
* **Автономность:** Возможность работы без постоянного присутствия оператора на всех ключевых этапах.
* **Модульность:** Лёгкое добавление новых функций и оборудования (например, сыроварня, второй котел).
* **Отказоустойчивость:** Наличие ручного и аварийного режимов для завершения цикла при сбоях автоматики.
* **Интеграция:** Обмен данными с вышестоящими системами построен по принципу **CBPi4 → Node-RED → внешние системы (MES/ERP, Home Assistant)**. Node-RED выполняет роль единого интеграционного ядра, получая данные от CBPi4 по MQTT, преобразуя их и направляя в MES/ERP. Взаимодействие с Home Assistant осуществляется через Node-RED (уровень упрощения определяется позднее). Прямое взаимодействие CBPi4 с Home Assistant исключено.

### 1.2. Режим "Пивоварение" (Производство сусла и брожение)

| Функция | Описание | Контролируемые параметры |
| :--- | :--- | :--- |
| **Затирание (Mashing)** | Автоматический нагрев и поддержание заданных температурных пауз в заторном чане (MLT) по настраиваемому профилю рецепта. Реализован HERMS-метод с использованием горячей воды из HLT1 и **противоточного теплообменника спирального типа (ПТТ)**. | Температура в MLT (TT-MLT), температура воды в HLT1 (TT-HLT1) |
| **Фильтрация и промывка (Lautering & Sparging)** | Автоматическая или ручная циркуляция сусла для осветления. Автоматическая подача горячей воды (75-78°C) из HLT2 для промывки дробины с контролем объема. | Температура в HLT2 (TT-HLT2), объем промывочной воды (расходомер) |
| **Кипячение сусла (Boiling)** | Автоматический нагрев сусла в котле (BK) до кипения. Поддержание активного кипения в течение заданного времени. Реализован **турбо-режим нагрева: до 95 °C нагрев осуществляется от HLT1 через противоточный теплообменник; выше 95 °C и на этапе кипячения нагрев переключается на ТЭН варочного котла (BK)**. Автоматические напоминания о внесении хмеля по таймеру. | Температура в BK (TT-BK) |
| **Охлаждение сусла (Cooling)** | Управление процессом охлаждения горячего сусла через противоточный теплообменник до температуры внесения дрожжей. | Температура сусла на выходе из чиллера |
| **Брожение (Fermentation)** | Поддержание заданной температуры в ферментере (ЦКТ) на всех этапах: главное брожение, диацетиловая пауза, дображивание, лагерное хранение. | Температура в ЦКТ (TT-F1) |

### 1.3. Режим "Винокурение" (Дистилляция и Ректификация)

| Функция | Описание | Контролируемые параметры |
| :--- | :--- | :--- |
| **Перегонка (Stripping)** | Быстрый нагрев куба с брагой для получения спирта-сырца. Управление мощностью нагрева без точной стабилизации. | Температура в кубе (TT-Boil) |
| **Ректификация (Rectification)** | Полное управление процессом получения высокоочищенного спирта. Включает: стабилизацию колонны, отбор голов, тела и хвостов. | Температура в кубе (TT-Boil), температура верха колонны (TT-ColTop), температура низа колонны (TT-ColMid) |
| **Стабилизация мощности** | Автоматическое поддержание заданной мощности нагрева куба (в Вт или %) независимо от колебаний напряжения в сети 220В. Критично для ректификации. | Мощность ТЭНа куба (косвенно: напряжение и ток) |
| **Управление флегмовым числом** | Автоматическое управление клапаном отбора дистиллята для поддержания заданного флегмового числа (например, пульсациями клапана). | |
| **Автоматическая смена фракций** | Алгоритм, который автоматически переключает приемные емкости или перенаправляет поток дистиллята при достижении заданных температурных границ (например, окончание отбора "голов" по температуре в кубе). | |

### 1.4. Общесистемные функции (для всех режимов)

| Функция | Описание |
| :--- | :--- |
| **Управление рецептами** | Под рецептом понимается не только перечень ингредиентов (солод, хмель, вода), но и полный управляющий сценарий, включающий: шаблон процесса (затирание / варка / дистилляция / ректификация); последовательность температурных пауз с временем выдержки; профили мощности нагрева; таймеры внесения хмеля (или переключения фракций); режимы работы насосов и клапанов. Рецепт хранится в CBPi4; возможны загрузка, редактирование, экспорт. |
| **Визуализация и мониторинг** | Отображение текущих параметров процесса (температуры, мощности, состояния клапанов) на **внешнем промышленном сенсорном мониторе 26″ (HDMI)**. Встроенного 7″ экрана на панели щита не предусмотрено. В составе щита необходимо обеспечить: 220В для питания монитора; HDMI для передачи видеосигнала; USB для сенсорного управления. |
| **Тренды и логирование** | Построение графиков изменения ключевых параметров в реальном времени. Запись всех событий и параметров в базу данных с возможностью экспорта. |
| **Сигнализация и уведомления** | Звуковая и световая сигнализация на щите управления. Отправка push-уведомлений в Telegram/Home Assistant о завершении этапов или аварийных ситуациях. |
| **Ручной режим (Manual)** | Возможность полностью отключить автоматику тумблером "ВЫСОКОЕ" и управлять ТЭНами, насосами и клапанами вручную с помощью кнопок и переключателей на передней панели щита. |
| **Резервный режим (Backup)** | При отказе основного контроллера (RPi) или потере связи, исполнительный контроллер (Mega) продолжает поддерживать последний заданный режим (например, держит температуру или мощность) для завершения цикла. |
| **ПАЗ (Противоаварийная защита)** | Автоматическое отключение всех ТЭНов и блокировка работы при срабатывании независимых датчиков: перегрев (105°C), протечка, превышение давления (для ПВК), аварийная кнопка "Стоп". **Защита от осушения ТЭНов** (снижение уровня воды или сусла) реализуется на независимых дискретных контроллерах, не связанных с основной автоматикой. Каждый ТЭН (HLT1, HLT2, BK) оснащается отдельным датчиком уровня, включённым в цепь ПАЗ. При срабатывании — разрыв цепи главного контактора и блокировка включения до ручного сброса. |
